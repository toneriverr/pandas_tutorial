第2回 | 拡張機能を使ったJupyter環境の作成¶

今回はPandas入門を進めていくために、Conda仮想環境を作成していきます。

具体的には、PandasライブラリのインストールとJupyter Notebookを使えるようにします。


さらに、今回はJupyterの拡張機能を使って、Pandasを学習しやすい環境にしていきます。

そこまで難しい話は登場しないので、サクッと環境構築を済ませていきましょう！

※WindowsおよびIntelMacでAnacondaをインストールしている場合は、今回のレクチャーをスルーしても大丈夫です！

Pandas入門のための環境構築¶

このレクチャーで行いたいのは、以下の3つです。

  • Conda仮想環境の作成
  • ライブラリのインストール
  • 拡張機能の導入

順番に進めていきましょう！

補足 : レクチャー開始前に¶

このPandas入門で使っているソースコードは、LINE公式アカウントで無料配布しています。

»参考リンク : LINE公式アカウント


もし動画を見ながら手を動かして学習していく場合には、ぜひソースコードをダウンロードしてから取り組んでみてください。

動画内ではソースコードをダウンロードしていることを前提に話を進めていきます。

Conda仮想環境の作成¶

まずはターミナルを立ち上げて、ダウンロードしたフォルダまで移動してあげます。

$ cd <ダウンロードしたフォルダ>


もしソースコードをデスクトップに置いたなら、以下のようになりますね。

$ cd Desktop/PandasBeginner


もしターミナル操作に慣れていなければ、cdを書いたあとにフォルダをドラック&ドロップすれば勝手にPATHを表示してくれます！

いずれかの方法で、いまダウンロードしてきたフォルダまで移動してあげましょう。


PandasBeginnerに移動したら、さっそくCondaの仮想環境を作成していきたいと思います。

Condaコマンドを使って仮想環境を作成するためには、以下のコマンドを入力します。

$ conda create -n <環境名> python=<Python Version>


今回はpandas_envという環境名、Pythonバージョンは3.8を使っていきたいと思います。

つまり、入力すべきコマンドは次のようになります。

$ conda create -n pandas_env python=3.8


これでEnterキーを入力しましょう。途中でy/nを聞かれますが、すべてy(yes)で答えてしまって大丈夫です。

コマンドの実行が完了するとpandas_envを作成できているかと思います。

※conda info -eを実行すると、作成済みのConda環境一覧を確認できます！


最後にいま作成した仮想環境に切り替えておきましょう。

そのためには、以下のコマンドを入力します。

$ conda activate pandas_env

これで作成した仮想環境に切り替えできました！

ライブラリのインストール¶

作成したpandas_envに切り替わっていることが確認できたら、ライブラリをインストールしていきます。

今回はとりあえず、Jupyter NotebookでPandasが使えれば良いので、以下のコマンドを入力してライブラリをインストールしていきます。

$ conda install pandas jupyter -y


ライブラリのインストールが完了したら、Jupyter Notebookを使ってみたいと思います。

ターミナルでjupyter notebookと入力して、Jupyter Notebookを立ち上げてみましょう！


おそらくJupyterのトップページにアクセスできるはずです！

拡張機能の導入¶

最後に拡張機能を導入していきましょう！

拡張機能を使うためには、jupyter_contrib_nbextensionsをインストールする必要があります。

ですが本テキストを作成している時点(=2021/5/5)で、最新のJupyter Notebookに対応する拡張機能は、Condaコマンドでインストールできないようです。


なのでcondaではなくpipを使ってインストールしていきましょう。

以下のコマンドを入力してjupyter_contrib_nbextensionsをインストールします。

$ pip install jupyter_contrib_nbextensions


インストールが完了したら、Jupyter Notebookを立ち上げましょう。

$ jupyter notebook

Jupyterのトップ画面にアクセスすると、新しく「Nbextensions」というタブが追加されているはずです。

あとは使っていきたい拡張機能にチェックを入れるだけです。

拡張機能はたくさんありますが、今回は以下の4つを使っていきたいと思います。

  • Table of Contents : 目次を追加する
  • Variable Inspector : 変数の一覧を表示する
  • Hinterland : 入力補完を使えるようにする
  • Codefolding : セル内のコードを折り畳めるようにする


基本的にはチェックを入れるだけで機能します。

HinterlandはVSCodeでMicrosoftが提供している「Python拡張機能」のように動くので非常に便利です！

ぜひコードを書いて試してみましょう！
